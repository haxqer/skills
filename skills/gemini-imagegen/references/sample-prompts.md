# Sample Prompts

## Product Hero

```text
Use case: product-mockup
Asset type: ecommerce hero
Primary request: a premium product photo of a matte black insulated bottle
Style/medium: editorial product photography
Composition/framing: centered bottle, slight three-quarter angle, generous negative space
Lighting/mood: soft daylight from the left, quiet premium mood
Constraints: no logo, no text, no watermark
Avoid: clutter, extra props, exaggerated reflections
```

## Landing Page Illustration

```text
Use case: stylized-concept
Asset type: SaaS landing page hero
Primary request: an optimistic illustration about distributed teamwork
Style/medium: clean editorial illustration
Composition/framing: wide composition with clear empty area for a headline on the right
Color palette: warm blue, sand, and coral accents
Constraints: no visible brand marks, no tiny unreadable UI text
```

## UI Mockup

```text
Use case: ui-mockup
Asset type: mobile app marketing mockup
Primary request: a polished mockup of a reading-tracker app on a phone
Style/medium: realistic device mockup
Composition/framing: angled phone on soft neutral background
Text (verbatim): "Read 24 min"
Constraints: modern typography, no watermark
Avoid: generic stock-photo background, purple glow
```

## Background Replacement

```text
Use case: precise-object-edit
Primary request: replace the background with a warm studio gradient
Constraints: change only the background; keep the product shape, color, and edge detail unchanged
Avoid: extra props, crop changes, added text
```

## Poster With Text

```text
Use case: infographic-diagram
Asset type: event poster
Primary request: a refined poster for a jazz night
Style/medium: print poster design
Composition/framing: bold headline at top, date block below, saxophone silhouette in lower half
Text (verbatim): "MIDNIGHT JAZZ"
Constraints: high legibility, strong contrast, premium editorial feel
Avoid: clip-art look, random extra letters, watermark
```
